bwidth

(./FEM/bwidth.m)

Function Synopsis

bw=bwidth(elems)

Help text


 BWIDTH compute the full bandwidth of an FEM element list

        INPUT: elems - 3 or 4 column element list; if the element list
                       is 4-column, BWIDTH assumes the first column is
                       an element counter and ignores it.

        OUTPUT: bandwidth

  bw=bwidth(elems)
