 OPNML/MATLAB5 routines not specifically related to FEM grid structures.
 Version 3.0
 24 July, 1998

 These are functions that are not directly related to FEM grids.

 AP_TO_RI_DEG   convert amplitude/phase(deg) vectors to real/imaginary.
 AP_TO_RI_RAD   convert amplitude/phase(rad) vectors to real/imaginary.
 BLANK          delete leading/trailing whitespace
 CIRCLES  plot  circles at optionally input centers with required input radii.
 CONTEXT_MENUS  script to create context menus
 ELLIPSE plot  ellipse given center, major/minor axes, orientation, etc. 
 ELLIPSE_EAST  plot ellipse given center, major/minor axes, orientation, etc. 
 ELLSCALE      draw a speed scale on an ellipse plot 
 GCP           get current point on current axes and current figure.
 GEN_DROG_GRID mouse-driven drogue input coordinate generator
 GEN_DROG_LINE mouse-driven drogue input coordinate generator
 ISDIR	        test to see if a directory exists
 ISINT         determine if input matrix is integers
 MARKCCW       mark Counter Clockwise oriented ellipses with *'s at centers
 MOVETEXT      grab and move text objects in the current axes
 NUMSCAL       label scalar (Q) values at x,y locations on current axes.
 PAN	        mouse-driven pan-around facility.   
 RI_TO_AP_DEG  convert real/imaginary vectors to amplitude/phase(deg)
 RI_TO_AP_RAD  convert real/imaginary vectors to amplitude/phase(rad)
 STRETCHBOX    create a box and stretch it following the mouse
 STRETCHLINE   create a line and stretch it following the mouse
 STRETCHLINE   create a line and stretch it following the mouse
 WYSIWYG       force printed output to be more like what one sees on screen
