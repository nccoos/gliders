blank

(./basics/blank.m)

Function Synopsis

outstr=blank(instr)

Help text

 BLANK strips leading and trailing whitespace and/or NULL characters
       from input string instr.  The MATLAB function DEBLANK only
       removes trailing blanks and NULLS.
