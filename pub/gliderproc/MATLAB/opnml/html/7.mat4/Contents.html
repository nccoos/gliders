 OPNML/MATLAB4 routines - provided for backward compatibility

 Provided to resolve existing references to MATLAB4 argument syntax:

 BELINT4    compute shape function information for a FEM domain
 DRAWELEMS4 draw FEM element configuration 
 EL_AREAS4  compute triangular finite element areas
 LCONTOUR2  obsolete; use lcontour4
 LCONTOUR4  contour a scalar field on a FEM grid.
 LOADGRID4  load principle gridfiles for a given domain name (input)
 PLOTBND4   plot boundary of FEM mesh
 VECPLOT2   routine to plot vectors.  

 Not yet ported, but still functional
 GENBEL4    build a boundary element file (.bel) for QUODDY.
 ISOPHASE   contour a vector of scalar (phase) values on a FEM grid.
 TELLIPSE   plot ellipses from amp/phase specification of a vel field
